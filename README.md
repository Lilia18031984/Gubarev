# 🎲 Random Task Generator

Приложение для генерации случайных задач с графическим интерфейсом на Tkinter.

## 👤 Автор
**Губарев Кирилл** 

## 📋 Описание

Random Task Generator — это десктопное приложение, которое позволяет:
- Генерировать случайные задачи из предопределённого списка
- Фильтровать задачи по типу (Учёба, Спорт, Работа, Другое)
- Добавлять свои собственные задачи
- Просматривать историю всех сгенерированных задач
- Сохранять данные между запусками программы

## 🚀 Установка и запуск

### Требования
- Python 3.6 или выше
- Tkinter (входит в стандартную библиотеку Python)
